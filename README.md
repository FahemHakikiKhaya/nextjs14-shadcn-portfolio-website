# Portfolio Website

![Portfolio](https://img.shields.io/badge/Portfolio-Website-blue.svg)

Welcome to my modern and minimalist portfolio website! This site showcases my work and skills with a focus on simplicity and elegance.

## Features

- **Modern Design**: A clean and simple layout powered by [Shadcn](https://shadcn.dev) and [Tailwind CSS](https://tailwindcss.com).
- **Interactive Animations**: Smooth animations created with [Framer Motion](https://www.framer.com/motion/).
- **Responsive Framework**: Built using [Next.js 14](https://nextjs.org) with the App Router for optimal performance and navigation.
